🤖 Auto-sync rule (goes in CLAUDE.md):
Whenever a skill is installed, created, or updated → copy the file to Claude Setup/Skills/ → commit + push.
Whenever a CARL domain is added or edited → copy to Claude Setup/CARL Domains/ → commit + push.
Whenever settings.json or global CLAUDE.md changes → copy to Claude Setup/Settings/ → commit + push.
✅ On a new machine: Clone the vault → copy Skills/ → CARL Domains/ → Settings/ to ~/.claude/ → done.
